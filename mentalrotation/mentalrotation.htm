Mental Rotation

You will be given two cube objects side by side.
The cubes can be the same or they can be mirror images of each other.


Are these cube objects the same object or are they mirror images?

In order to decide whether the cube objects are the same or mirror images, you can mentally rotate one of these objects. For example, if you are able to rotate the right object into a position to match the cube object on the left then the objects are the same. If you cannot, then the objects are different.

Try it out. Continue to see if you are correct.
